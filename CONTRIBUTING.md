# Contributing to Social Poster : pxltr Studio

We appreciate your interest in making this tool better for everyone. Please follow these guidelines to ensure a smooth collaboration.

## Ways to Help

* Style Innovation: Propose new color palettes by adding CSS classes to the reactive engine.
* Layout Testing: Help us test the 1:1 safety margins on different devices like tablets or foldable phones.
* Feature Requests: Suggest new ways to make the tool more useful for creators.

## Process

1. Fork the Repository: Head over to the project page and create your own branch to work on.
2. Code Standards: Keep the code clean and use Tailwind CSS for styling whenever possible.
3. Documentation: Ensure any new features are explained in the README.
4. Pull Requests: Submit your changes for review. Provide a clear description of what was added or fixed.

## Theme Contribution Example

If you want to add a new background vibe, you can add a keyword to the script and a corresponding CSS class. For example, a "Midnight" theme could be triggered by words like "Night" or "Dark."
